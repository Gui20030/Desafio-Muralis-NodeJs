Esta é a Rest API feita em Node.Js feita para o Deafio Muralis - Estágio além do sistema!

Essa API foi desenvolvida para registro de despesas.

|

Como utilizar a API:

|

Ao acessar a aba de registro da API em: localhost:3000/api/depesas

Insira os dados de acordo com o padrão a seguir:

|

"id": *Número de ID da compra*

"valor": *Valor da compra*,

"data_compra": "21/09",

"descricao": *Descrição da compra*,

"tipo_pagamento_id": *ID do tipo de pagamento* (1: Dinheiro, 2: Débito, 3: Crédito e 4: Pix),

"categoria_id": *ID de categoria da compra*
